# Liga Basquet Web

Base publica del proyecto web para la Liga Social Femenina de Basquetbol.

## Stack

- React
- Vite
- TypeScript
- React Router
- Supabase JS
- CSS
- Deploy pensado para Vercel

## Scripts

- `npm run dev`
- `npm run build`
- `npm run preview`

## Alcance actual

Esta primera etapa deja lista la estructura del proyecto, el layout publico, las rutas base y placeholders visuales elegantes para empezar a conectar datos reales en la siguiente fase.
